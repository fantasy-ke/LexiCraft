# TypeWords 主页设计说明

## 设计概述

新的主页采用现代化的蓝白色简约设计风格，符合英语学习应用的专业性和友好性要求。

## 设计特点

### 🎨 视觉设计

- **配色方案**: 蓝白色渐变背景，体现清新简约的风格
- **动画效果**: 浮动几何形状背景动画，增加视觉趣味性
- **响应式设计**: 完美适配桌面端和移动端

### 🧭 导航结构

- **固定头部**: 半透明毛玻璃效果，包含Logo和右侧按钮
- **Logo设计**: TW字母标识 + TypeWords文字
- **右侧按钮**: 登录按钮 + GitHub图标链接

### 📱 功能展示

- **Hero区域**: 大标题 + 副标题 + 描述文字
- **功能卡片**: 三个主要功能模块的介绍
    - 单词练习：四种练习模式，智能记忆曲线
    - 文章背诵：逐句练习，语音朗读
    - 学习统计：详细的学习数据分析
- **CTA按钮**: 醒目的"开始学习"按钮

## 技术实现

### 🛠 技术栈

- Vue 3 + TypeScript + Composition API
- UnoCSS 原子化CSS
- 自定义CSS动画和过渡效果

### 🎯 动画效果

- **淡入上升动画**: 内容逐步显示，增强用户体验
- **浮动背景**: 几何形状的缓慢浮动动画
- **悬停效果**: 卡片和按钮的交互反馈
- **无障碍支持**: 支持 `prefers-reduced-motion` 媒体查询

### 🔗 路由配置

- **主页路径**: `/` - 新的欢迎页面
- **应用路径**: `/app/*` - 原有的应用功能
- **兼容性**: 旧路由自动重定向到新路径

## 用户体验

### 🎪 交互流程

1. 用户访问网站首页，看到欢迎界面
2. 浏览功能介绍，了解应用特色
3. 点击"开始学习"按钮进入应用
4. 进入应用后显示侧边栏等完整界面

### 📐 设计原则

- **简约优先**: 去除不必要的视觉元素
- **内容为王**: 突出核心功能和价值主张
- **渐进式体验**: 从简单的欢迎页到功能丰富的应用
- **性能优化**: 轻量级动画，快速加载
